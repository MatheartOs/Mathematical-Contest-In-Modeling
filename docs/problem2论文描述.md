# Problem 2 论文描述：新流入文件迁移分类与综合评价

> 本文档用于支撑数维杯 B 题论文中“问题二：新流入数据归属判断与综合评价”部分的撰写。内容对应当前实现 `src/mcm_b/problem2_transfer.py`、运行脚本 `scripts/run_problem2_transfer.py`，以及输出目录 `outputs/b_problem/problem2_transfer/`。论文手可直接据此改写为论文正文、公式说明、算法流程、评价指标和结果分析。

## 1. 问题二目标

问题二要求在问题一已经建立的历史文件分类体系基础上，对后续流入的新数据进行迁移归属判断。数据集 2 和数据集 3 与数据集 1 存在明显差异：

1. 数据集 1 是历史真实文件，文本和结构信息相对完整，适合作为源域建立主题体系。
2. 数据集 2 是后续流入的半结构化记录，字段相对规整，但内容常表现为标题、片段、表格行等不完整形式。
3. 数据集 3 是匿名原始文件或行记录，来源信息更弱，主题边界更模糊，迁移分类风险更高。

因此，问题二不能只输出一个类别标签，还必须回答四个问题：

1. 新文件属于哪个历史主题类别。
2. 该归属是否合理，是否存在主次类别接近的情况。
3. 该归属是否可解释，模型是否能指出支撑分类的主题词。
4. 数据集 2、3 与历史源域之间的迁移适用性如何。

本项目采用“异构图主题空间复用 + 距离概率化分类 + ARS 归属合理性 + MII 可解释性 + MMD/TAI 迁移适用性 + A/B/C 人机协同状态判别”的方法解决问题二。

## 4. 统一建模文本构造

每个文件的建模文本由标题、正文和业务关键词共同构成：

```text
model_text_i = title_i + clean_text_i + business_keywords_i
```

之后执行中文语义过滤：

1. 删除英文、数字、URL、编号和小数形式。
2. 只保留中文字符。
3. 删除模板噪声词，如“工作表名称”“表头字段”“主要关键词”“样例内容”“图片名称”“图片编号”“下载”“关键词”。

记第 `i` 个文件的建模文本为：

```text
d_i = Clean(title_i, clean_text_i, business_keywords_i)
```

这一处理保证迁移分类主要依据业务语义和主题共现结构，而不是被表格编号、文件编码、URL 或 OCR 数字噪声影响。

## 5. 源域异构图主题空间复用

问题二不是重新发现类别，而是复用问题一在数据集 1 上得到的主题体系。因此，源域仍采用问题一的异构图表征方法。

### 5.1 文档-词 TF-IDF 边

采用字符级 2-4 gram 作为中文词项，不依赖外部分词工具。设源域共有 `N` 个文档，词表大小为 `M`。文档-词矩阵为：

```text
X_tfidf ∈ R^{N × M}
```

其中第 `i` 个文档、第 `j` 个词项的权重为：

```text
tfidf_{ij} = tf_{ij} * log(N / df_j)
```

实际参数为：

| 参数 | 当前值 | 含义 |
| --- | ---: | --- |
| `ngram_range` | `(2, 4)` | 中文 2 至 4 字符片段 |
| `min_df` | `2` | 至少出现在 2 个历史文档中 |
| `max_df` | `0.88` | 过滤过泛词项 |
| `max_features` | `2500` | 词项上限 |
| `sublinear_tf` | `True` | 使用对数 TF 缓解长文本支配 |

最终词项数为：

```text
term_count = 2500
```

### 5.2 词-词正 PMI 图

为了刻画词项之间的共现关系，模型在源域文本上使用滑动窗口统计词项共现，并计算正 PMI。设 `n_W(a,b)` 为词项 `a` 与 `b` 在滑动窗口中共同出现的次数，`n_W(a)` 与 `n_W(b)` 为各自出现的窗口数，`N_W` 为窗口总数，则：

```text
PMI(a,b) = log( n_W(a,b) * N_W / (n_W(a) * n_W(b)) )
```

仅保留正 PMI：

```text
W_ab = max(PMI(a,b), 0)
```

并对词图矩阵按行归一化，得到词-词传播矩阵：

```text
W_ppmi
```

当前运行得到：

```text
word_graph_edges = 366841
word_graph_windows = 404135
```

### 5.3 两跳图传播

问题一解析文档建议使用 GCN 聚合。当前实现采用不依赖深度学习框架的轻量图传播近似。对文档-词矩阵进行两跳词图传播：

```text
X_graph = X_tfidf + 0.60 * X_tfidf W_ppmi + 0.30 * X_tfidf W_ppmi^2
```

其中：

1. `X_tfidf` 保留文档自身关键词。
2. `X_tfidf W_ppmi` 引入一跳共现词。
3. `X_tfidf W_ppmi^2` 引入二跳语义邻域。

该步骤等价于把词图中的局部共现结构传播到文档表示中，使新文件即使没有完全命中历史主题词，也能通过共现邻域获得相近主题表达。

## 6. 结构与业务特征融合

仅使用文本可能无法充分表示多源异构文件的形式差异。模型进一步融合结构特征和业务特征。

结构特征包括：


业务特征包括：

```text

has_money
has_contract
has_personnel
has_deadline
has_urgent
```

记结构业务特征矩阵为：

```text
X_meta
```

先在源域上拟合标准化器，再对源域和目标域使用同一标准化变换：

```text
X_meta' = StandardScaler(X_meta)
```

随后进行特征拼接：

```text
X_fused = [X_graph, X_meta']
```

再进行 L2 归一化：

```text
X_norm = Normalize(X_fused)
```

## 7. 共享低维迁移空间

为了降低稀疏高维矩阵的噪声，模型在源域上拟合 TruncatedSVD，并将源域和目标域投影到同一低维空间：

```text
Z = SVD(X_norm)
```

当前嵌入维度为：

```text
embedding_dim = 80
```

之后再次进行标准化：

```text
Z' = StandardScaler(Z)
```

该共享空间同时包含文本主题、词图共现、结构质量和业务属性信息，是后续迁移分类与分布差异评价的基础。

## 8. 源域聚类与目标域归属

在源域嵌入 `Z_s` 上使用 KMeans 得到 10 个历史主题中心：

```text
{c_1, c_2, ..., c_K}, K = 10
```

对于目标域第 `i` 个文件，其嵌入向量为 `z_i`，计算到各主题中心的欧氏距离：

```text
d_{ik} = || z_i - c_k ||_2
```

距离越小，说明文件越接近该历史主题。为了获得类别概率分布，模型使用温度 softmax 将距离转为概率：

```text
p_{ik} = exp(-d_{ik} / \tau) / sum_{r=1}^{K} exp(-d_{ir} / \tau)
```

其中温度参数由源域样本到最近中心距离的中位数自适应给出：

```text
\tau = 0.12 * median(min_k d_{sk})
```

这样可以避免所有类别概率过于平均，使概率分布更能反映主次类别差异。

目标样本的预测主题为：

```text
y_i = argmax_k p_{ik}
```

同时记录概率最高的两个主题：

```text
p_{i1} = max_k p_{ik}
p_{i2} = second_max_k p_{ik}
```

这两个概率用于后续判断归属是否清晰、是否存在多类别重叠。

### 8.1 语义锚定概率校准

在初版迁移实验中发现，数据集 3 的短行记录容易受到部分历史主题中心吸附。当某一主题的 c-TF-IDF 词中含有“服务”“中心”“项目”等泛化词时，单纯依赖向量距离可能造成类别偏置。为避免该问题，最终模型加入语义锚定概率校准。

对于每个历史主题，模型保留该主题的 c-TF-IDF 代表词，同时补充人工可解释的主题锚词。例如：

```text
教育教学管理类：教学、教师、学生、课程、教育、学院、本科、招生、推免
养老服务机构类：老年、养老、养老院、老年公寓、公寓、康养、适老
生态环境治理类：生态、环境、污染、水质、能源、低碳、绿色、排放
制造业产业统计类：制造、制造业、工业、生产、装备、高技术、规模以上
```

同时过滤“服务”“中心”“单位”“项目”“工作”“管理”“研究”等泛化词，避免它们单独支配主题归属。设第 `i` 个目标文件对第 `k` 个主题的锚词命中得分为：

```text
s_{ik} = sum_{t in A_k} count(t in d_i) * w_t
```

其中 `A_k` 为第 `k` 个主题的有效锚词集合，`w_t` 为词项权重。对得分进行平滑归一化得到语义锚定概率：

```text
q_{ik} = SmoothNormalize(log(1 + s_{ik}))
```

最终类别概率由图空间距离概率和语义锚定概率加权融合：

```text
\mathbf{p}_i = (1 - \beta)\mathbf{p}_i^{dist} + \beta\mathbf{p}_i^{lex}
```

当前取：

```text
\beta = 0.45
```

该校准保持问题一的源域主题体系不变，只对目标域归属概率进行语义纠偏。其作用是让短文本样本更多依据明确业务词归属，而不是被结构相似或泛词相似的历史中心吸附。

## 9. 归属合理性指标 ARS

问题二要求评价分类结果的合理性。本文定义归属合理性分数 ARS（Assignment Reasonableness Score）：

```text
ARS_i = \alpha p_{i1} + (1 - \alpha) * (p_{i1} - p_{i2}) / p_{i1}
```

其中：

```text
\alpha = 0.60
```

该指标由两部分组成：

1. `p_{i1}` 表示模型对第一类别的绝对置信度。
2. `(p_{i1} - p_{i2}) / p_{i1}` 表示第一类别相对第二类别的优势。

若 `p_{i1}` 较高且主次类别差距较大，则 `ARS_i` 较高，说明归属更合理。若第一、第二类别概率接近，即使最高概率不低，也会被判为边界样本。

## 10. 信息熵与不确定性

对类别概率向量 `\mathbf{p}_i = [p_{i1}, ..., p_{iK}]` 计算归一化信息熵：

```text
H_i = - sum_{k=1}^{K} p_{ik} log_2(p_{ik}) / log_2(K)
```

其中 `H_i ∈ [0,1]`：

1. `H_i` 越小，说明概率集中于少数类别，分类更清晰。
2. `H_i` 越大，说明概率分散，可能存在多类别重叠或分布外样本。

该指标与 `ARS` 配合，用于识别边界样本和未知样本。

## 11. 可解释性指标 MII

问题二还要求评价分类的可解释性。由于当前实现没有使用深度学习注意力层，模型采用“主题词贡献集中度”构造 MII（Model Interpretability Index）。

对于每个主题，问题一已经通过 c-TF-IDF 得到主题词集合：

```text
T_k = {t_1, t_2, ..., t_{L_e}}
```

对于目标文件 `d_i`，若其预测主题为 `k`，则统计主题词在目标文本中的命中贡献：

```text
s_j = count(t_j in d_i) * len(t_j)
```

将贡献归一化为：

```text
a_j = s_j / sum_{r=1}^{L_e} s_r
```

若主题解释由少数高贡献主题词支撑，则解释更集中；若贡献分散，则解释性较弱。定义：

```text
MII_i = 1 - ( - sum_{j=1}^{L_e} a_j log_2(a_j) ) / log_2(L_e)
```

其中 `MII_i ∈ [0,1]`：

1. `MII_i` 越接近 1，说明少数主题词即可明确解释该归属。
2. `MII_i` 越接近 0，说明主题词贡献分散或缺少明显解释词。

输出表中的 `explanation_terms` 字段记录具体命中的解释词及其贡献，例如：

```text
教学:12 / 项目:8 / 教育:6
```

## 12. 迁移适用性指标 MMD 与 TAI

为了评价数据集 2、3 与历史源域之间的分布差异，模型使用最大均值差异 MMD（Maximum Mean Discrepancy）。

设源域嵌入样本为：

```text
Z_s = {z_1^s, z_2^s, ..., z_{n_s}^s}
```

目标域嵌入样本为：

```text
Z_t = {z_1^t, z_2^t, ..., z_{n_t}^t}
```

下文记源域样本量为 `n_s`，目标域样本量为 `n_t`。

采用 RBF 核函数：

```text
k(x,y) = exp(-\gamma ||x-y||^2)
```

其中：

```text
\gamma = 1 / embedding_dim
```

MMD 平方定义为：

```text
MMD^2(Z_s, Z_t)
= 1/n_s^2 sum_i sum_j k(z_i^s,z_j^s)
+ 1/n_t^2 sum_i sum_j k(z_i^t,z_j^t)
- 2/(n_s n_t) sum_i sum_j k(z_i^s,z_j^t)
```

为了便于解释，将 MMD 转换为迁移适用性指数 TAI（Transfer Applicability Index）：

```text
TAI_d = exp(-MMD^2(Z_s, Z_t^{(d)}))
```

`TAI ∈ (0,1]`，值越大表示目标域与源域分布越接近，迁移适用性越好。

当前结果为：

| 数据集 | TAI | 解释 |
| --- | ---: | --- |
| dataset2 | 0.588393 | 半结构化数据字段规整，但文本主题较集中，迁移适用性中等 |
| dataset3 | 0.699263 | 匿名记录在锚点增强空间中与源域主题更接近，迁移适用性较高 |

## 13. 多类别重叠与未知样本识别

根据题目要求，对于多类别重叠样本和无法明确归类样本，需要建立识别模型并给出处理思路。本模型将目标文件分为四种状态。

下列规则中，`m_i = (p_{i1}-p_{i2})/p_{i1}` 表示主次主题相对边际差，`Q_i` 表示样本解析质量。

### 13.1 清晰自动归档

若样本满足以下条件：

```text
p_{i1} >= 0.30
ARS_i >= 0.30
m_i >= 0.12
H_i <= 0.90
MII_i >= 0.03
```

则判定为：

```text
A_clear_auto_archive
```

含义为：主类别概率较高，主次类别差距明显，概率分布较集中，并且可以找到一定主题词解释，因此可直接自动归档。

### 13.2 辅助归档/抽检

若样本不属于明显重叠或未知，但未达到清晰自动归档的严格阈值，则判定为：

```text
A_assisted_archive
```

含义为：系统可以给出归属类别，但建议保留解释信息，后续可按比例抽检。

### 13.3 多类别重叠复核

若满足：

```text
m_i < 0.08
```

或：

```text
p_{i1} < 0.22 且 p_{i1} - p_{i2} < 0.025
```

则判定为：

```text
B_overlap_manual_review
```

含义为：第一类别和第二类别概率接近，文件可能同时涉及多个业务主题，需要人工复核其最终归档类别。

### 13.4 未知专家研判

若满足：

```text
p_{i1} < 0.16
```

或：

```text
H_i > 0.97
```

或：

```text
Q_i < 0.35
```

或文件无法形成稳定文本表征，则判定为：

```text
C_unknown_expert_review
```

含义为：该文件可能属于源域主题体系之外的新类别，或清洗质量不足以支持自动归档，需要专家研判。

## 19. 论文可誊写的算法流程

可在论文中将问题二算法写为以下步骤。

### 算法 2：基于异构图主题空间的迁移分类与评价算法

输入：

```text
历史源域文件画像 D_s
新流入目标域文件画像 D_t
问题一主题数 K
```

输出：

```text
目标文件类别 y_i
类别概率向量 `\mathbf{p}_i`
ARS、MII、TAI
自动归档/复核状态 state_i
```

步骤：

1. 对源域和目标域文件构造统一建模文本 `d_i`。
2. 在源域上训练字符级 TF-IDF，得到文档-词矩阵 `X_tfidf`。
3. 在源域文本上统计滑动窗口共现，计算正 PMI 词图 `W_ppmi`。
4. 对源域和目标域执行两跳图传播，得到 `X_graph`。
5. 融合结构特征和业务特征，得到 `X_fused`。
6. 在源域上拟合 SVD、标准化器和 KMeans 主题中心。
7. 将目标域投影到同一嵌入空间，计算到各主题中心的距离。
8. 使用温度 softmax 将距离转换为类别概率分布。
9. 计算主类概率、主次类差、ARS、熵和 MII。
10. 使用 MMD 计算数据集级迁移适用性 TAI。
11. 根据阈值规则将样本划分为清晰自动归档、辅助归档、多类别重叠复核、未知专家研判。
12. 输出逐文件分类表、数据集级评价表、边界样本表和图表数据。

## 22. 局限性与改进方向

当前问题二模型具有较好的可解释性和可复现性，但仍存在以下局限：

1. 当前类别体系完全继承问题一，若数据集 2 或 3 出现历史源域不存在的新主题，模型只能识别为未知专家研判，不能自动新建类别。
2. 当前 MII 使用主题词贡献近似注意力解释，若后续引入 BERT 或 GCN，可用真实注意力或梯度归因进一步增强解释性。
3. 当前 TAI 是数据集级指标，若需要更精细控制，可扩展为单文件局部 MMD 或最近邻密度评分。
4. 数据集 3 中部分行记录文本较短，主题概率可能受到结构特征影响较大，后续可增加字段级语义增强。

论文中可将这些作为模型改进方向，而不影响当前问题二结果的完整性。

## 23. 可直接引用的结论段

本研究在问题一历史主题体系基础上，构建了面向新流入数据的迁移分类模型。模型首先复用数据集 1 上建立的文本-单词异构图主题空间，通过 TF-IDF 文档-词边、正 PMI 词-词边和两跳图传播获得语义增强表示；随后融合文件结构特征、清洗质量特征和业务标签，将数据集 2、3 投影到同一低维嵌入空间，并根据目标样本到历史主题中心的距离计算类别概率分布。为了避免单纯硬分类带来的误判风险，进一步设计了 ARS 归属合理性、MII 可解释性和 TAI 迁移适用性指标，并将样本划分为清晰自动归档、辅助归档、多类别重叠复核和未知专家研判四类状态。

实验结果表明，模型共处理 4519 条新流入记录，其中 4419 条可形成稳定迁移表征。数据集 2 的迁移适用性指数为 0.588393，数据集 3 的迁移适用性指数为 0.699263，说明业务锚点增强后匿名记录中可解释主题结构更明显。经语义锚定概率校准和治理阈值重校准后，数据集 2 中 787 条记录可自动或辅助归档，134 条进入多类别重叠复核，80 条进入未知专家研判；数据集 3 中 3263 条可自动或辅助归档，233 条进入多类别重叠复核，22 条进入未知专家研判。数据集 3 的主题分布由初版的单类吸附修正为教育、医药审批、生态、财政、项目等多类并存，更符合抽样文本所反映的真实业务结构。该结果说明模型能够在完成新文件分类的同时识别边界样本和未知样本，为后续人工复核优先级排序和资源约束优化提供了可靠输入。
