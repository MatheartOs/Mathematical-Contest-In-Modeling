## 附录 1：支撑材料列表

### 1.1 实际使用的软件与运行环境

| 软件或工具 | 作用 |
| --- | --- |
| Windows PowerShell | 执行项目运行命令、清理输出目录、查看结果文件 |
| Python 3.10 虚拟环境 `.venv` | 运行全部数据清洗、建模、优化与结果汇总程序 |
| pandas | 读取和写出 CSV、Excel 表格，完成数据汇总与字段合并 |
| numpy | 数值计算、矩阵运算、评分归一化 |
| scipy | 稀疏矩阵运算，支撑 TF-IDF 与词图传播 |
| scikit-learn | TF-IDF、标准化、TruncatedSVD、KMeans、聚类指标计算 |
| matplotlib | 生成问题二、问题三及最终结果图 |
| python-docx | 读取 Word 文档文本、表格与段落结构 |
| openpyxl | 读取 Excel 文件与数据集 4 资源约束表 |
| PaddleOCR / PaddlePaddle GPU | 对图片文件执行 OCR 文字识别 |
| Git | 管理源程序版本与提交记录 |

### 1.2 项目运行命令

| 命令 | 作用 |
| --- | --- |
| `$env:PADDLEOCR_DEVICE='gpu:0'; .\.venv\Scripts\python.exe scripts\run_b_cleaning.py --output-dir outputs\b_problem\cleaning_ocr_full --max-chars 30000 --max-file-mb 25` | 从原始多源文件开始执行全量数据清洗、OCR、质量评估和人工复核清单生成 |
| `.\.venv\Scripts\python.exe scripts\recalibrate_cleaning_review.py --cleaning-dir outputs\b_problem\cleaning_ocr_full` | 根据资源约束重新校准清洗阶段人工复核清单 |
| `.\.venv\Scripts\python.exe scripts\run_problem1_innovative.py --cleaning-dir outputs\b_problem\cleaning_ocr_full --output-dir outputs\b_problem\problem1_innovative --clusters 10 --max-terms 2500` | 运行问题一历史文件主题分类与主题归纳模型 |
| `.\.venv\Scripts\python.exe scripts\run_problem2_transfer.py --cleaning-dir outputs\b_problem\cleaning_ocr_full --output-dir outputs\b_problem\problem2_transfer --clusters 10 --max-terms 2500` | 运行问题二新流入数据迁移归属、合理性评价和状态识别模型 |
| `.\.venv\Scripts\python.exe scripts\run_problem3_optimization.py --cleaning-dir outputs\b_problem\cleaning_ocr_full --classification-path outputs\b_problem\problem2_transfer\problem2_transfer_classification.csv --output-dir outputs\b_problem\problem3_optimization` | 运行问题三复核优先级评分与资源约束优化模型 |
| `.\.venv\Scripts\python.exe scripts\build_final_deliverables.py` | 汇总生成论文正文建议使用的五类最终结果表和预处理图 |
| `.\.venv\Scripts\python.exe -m unittest discover -s tests` | 运行单元测试，验证核心程序可执行性 |

### 1.3 源程序文件列表

| 文件名 | 作用 |
| --- | --- |
| `src/mcm_b/cleaning.py` | 多源异构数据清洗主程序；完成文件扫描、TXT/Word/Excel/PDF/图片解析、PaddleOCR 识别、文本标准化、字段抽取、质量评分和清洗阶段复核清单生成 |
| `src/mcm_b/problem1_innovative.py` | 问题一核心模型；构建文本-单词异构图、PMI 词图、两跳图传播、业务锚点增强、KMeans 聚类和 c-TF-IDF 主题解释 |
| `src/mcm_b/problem2_transfer.py` | 问题二核心模型；复用问题一主题空间，对数据集 2、3 进行迁移归属，计算 ARS、MII、TAI，并识别自动归档、重叠复核和未知研判状态 |
| `src/mcm_b/problem3_optimization.py` | 问题三核心模型；构建紧急程度、错分风险、复核必要性和综合优先级评分，结合资源约束生成复核队列与自动归档队列 |
| `src/mcm_b/paths.py` | 统一管理数据目录和输出目录路径 |
| `src/mcm_b/readers.py` | 早期通用文件读取工具，包含 TXT、Word、Excel、PDF、图片元数据读取函数 |
| `src/mcm_b/features.py` | 早期文件特征构造和主题关键词统计工具 |
| `src/mcm_b/modeling.py` | 早期 TF-IDF/KMeans 主题建模、分类和模型保存工具 |
| `src/mcm_b/risk.py` | 早期复核风险评分和资源场景分配工具 |
| `src/mcm_b/__init__.py` | Python 包初始化文件 |
| `scripts/run_b_cleaning.py` | 数据清洗命令行入口 |
| `scripts/recalibrate_cleaning_review.py` | 清洗阶段人工复核资源感知重校准脚本 |
| `scripts/run_problem1_innovative.py` | 问题一模型运行入口，写出主题分类结果、主题摘要和聚类指标 |
| `scripts/run_problem2_transfer.py` | 问题二模型运行入口，写出迁移分类结果、评价表、边界样本和图表 |
| `scripts/run_problem3_optimization.py` | 问题三模型运行入口，写出复核优先级表、资源场景对比表和图表 |
| `scripts/build_final_deliverables.py` | 最终交付表构建脚本，生成解析文档建议的五类核心结果表 |
| `scripts/run_b_pipeline.py` | 早期一体化流水线脚本，用于从清洗到建模的整体实验 |
| `scripts/run_b_pipeline_sample.py` | 早期抽样流水线脚本，用于小样本快速调试 |
| `scripts/inspect_b_data.py` | 数据结构检查和样本预览脚本 |
| `requirements.txt` | Python 依赖包列表 |
| `README.md` | 项目运行说明 |
